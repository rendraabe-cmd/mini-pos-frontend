# Mini POS - Frontend

Frontend aplikasi Mini POS (Point of Sale) menggunakan Vue.js 3.

## 🚀 Tech Stack

- **Vue.js 3** (Composition API)
- **Vite** (Build Tool)
- **Vue Router 4** (Routing)
- **Pinia** (State Management)
- **Tailwind CSS 3** (Styling)
- **Axios** (HTTP Client)

## ✨ Features

- 🔐 **Authentication** - Login dengan Sanctum Bearer Token
- 📊 **Dashboard** - Stats penjualan, top products, recent transactions
- 🛒 **POS / Kasir** - Cart system, payment, auto-receipt
- 📦 **Products CRUD** - Manage produk dengan kategori
- 🏷️ **Categories CRUD** - Manage kategori produk
- 💰 **Transactions History** - Riwayat lengkap + detail receipt

## 🔧 Setup

\`\`\`bash
npm install
npm run dev
npm run build
\`\`\`
